# Road-Segmentation

## Download 
1.Download dataset from http://www.cvlibs.net/datasets/kitti/eval_road.php

2.Unzip zip file

3.Rename the data folder 
```
-- data_road\ 
    -- training\
       -- image
       -- label
       -- label_grayscale
    -- testing\
       -- image
       -- label

```
## Run script

```
Run convert.py
Run main.py train 
```

## Requirement

```
pytorch (cuda supported)

```
